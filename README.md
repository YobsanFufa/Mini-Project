# Mini Project: OOP-Based Todo App

A simple command-line Todo application built with Python.

## Features

- Add tasks
- List tasks
- Complete tasks
- Delete tasks
- Data persistence via JSON

## Usage

### Add a Task

`python main.py add --title "Buy Milk" --desc "2% fat"`

### List Tasks

`python main.py list`

### Complete a Task

`python main.py complete --index 1`

### Delete a Task

`python main.py delete --index 1`
